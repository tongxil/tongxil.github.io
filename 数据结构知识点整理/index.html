# 线性表的定义：

由 n (n>= 0) 个元素组成的有序序列

# 线性表的顺序存储（相当于数组）

线性表（a1 a2 a3 …… an）

# 线性表的链式存储（链表）

链表：由多个结点组成的数据结构称为链表

结点：由数据域和指针域组成，数据域用来存储真实的数据，指针域用来存储下一个结点的地址

头指针与头结点（头结点的数据域 * 可以 ** 不 * 存储任何信息，其指针域存储指向第一个结点的指针（即指向头指针）：

​ 头结点：一般把链表的第一个结点称为头结点，头结点的数据域可以不存储数据，指针域存储下一个结点的地址（也是头指针）
